## Getting Started

Clone the repository

```bash
git clone https://github.com/reetaisync/reetai-website.git
cd reetai-website
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

or

```bash
npm start
```

The website will be available at:

```
http://localhost:3000
```
